# A-Level CIE Study Master

## 项目概述
A-Level CIE 学习助手 - 数学(9709) + 物理(9702) 完整课程学习平台

## 功能特性
- 📚 完整课程大纲 (AS + A2)
- 🤖 AI智能生成练习题
- 📤 文件导入习题集
- 📊 学习进度追踪
- 🌐 中英文对照

## 技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- Tailwind CSS (CDN)
- 纯前端实现，无需后端

## 项目结构
```
alevel-study-app/
├── index.html              # 主入口文件
├── css/
│   └── styles.css         # 样式文件
├── js/
│   ├── data/
│   │   └── questions.js   # 题库数据
│   └── main.js            # 主逻辑
├── CODE_STANDARDS.md      # 代码规范
└── README.md              # 项目说明
```

## 开发规范
- 单文件不超过500行
- 函数不超过50行
- 使用语义化命名
- 添加必要注释
