# A-Level CIE 学习助手 - 产品规格书

## 📋 项目概述

**项目名称：** A-Level CIE Study Master  
**类型：** 网页应用（Web App）  
**目标用户：** A-Level CIE 考生  
**核心功能：** 每日一课 + 智能练习 + 个性化学习路径  

---

## 🎯 功能模块

### 1. 每日一课 (Daily Lesson)
- 📚 每日推送一个知识点（中英文对照）
- 🎯 考试重点、难点、易错点清晰标注
- 📝 知识讲解 + 图表说明

### 2. 视频/资料推荐
- 🎬 嵌入 B站、知乎、CIE 官方资源链接
- 📝 每条链接带简短备注说明
- 🔗 点击可直接跳转

### 3. 智能练习
- 📋 从 CIE 题库抽取题目
- 📅 标注考试年份（如 June 2023 Paper 3）
- 🔄 根据错题自动推荐强化练习

### 4. 错题本 & 智能分析
- 📤 支持导入 PDF 习题集/错题集
- 🔍 自动识别知识薄弱点
- 📊 可视化掌握情况雷达图

### 5. 学习总结
- 💡 重点概念清单
- ⚠️ 易混淆概念对比表
- 📈 进度追踪

---

## 🎨 UI 设计风格

**苹果风格 (Apple Design)：**
- 🏠 圆角卡片 (border-radius: 16px)
- 🎨 柔和渐变背景
- ✨ 毛玻璃效果 (backdrop-filter: blur)
- 🔤 SF Pro 风格字体
- 📱 简洁留白，大字号
- 🌈 莫兰迪色系

---

## 🛠 技术方案

### 前端
- 单页应用 (SPA)
- React / Vue.js
- Tailwind CSS（实现苹果风格）

### 后端（可选）
- Python Flask/FastAPI
- 本地题库 JSON 文件
- PDF 解析（PyPDF2）

### 数据存储
- LocalStorage（浏览器本地）
- 或 JSON 文件

---

## 📦 第一阶段 MVP 功能

1. **首页** - 每日知识点展示（中英双语）
2. **练习页** - 选择科目→做题→查看答案
3. **导入页** - PDF 导入框（模拟）
4. **进度页** - 学习统计
